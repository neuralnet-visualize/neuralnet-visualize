Neural Network Visualizer documentation!
==========================================

Neural Network Visualizer is an extension of graphviz which can be used to create nice and publication ready images of neural networks.

Getting Started
---------------

1. Check out the :doc:`quickstart`
2. Star us on `GitHub <https://github.com/AnuragAnalog/nn_visualize>`_

Contributing
------------
Want to contribute? But don't know where to start, we have got a list of ways in which you can contribute.

we follow `Numpy Doc Style Guide <https://numpydoc.readthedocs.io/en/latest/>`_

1. Bug Reports
2. Feature Requests
3. Bug fixes
4. Add Example Notebook
5. Documentation improvements

Thank you for your contributions!!!
